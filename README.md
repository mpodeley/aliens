# HERCULANEUM-299

Un ensayo especulativo sobre física del tiempo, consolidación intersubjetiva del pasado, y fenomenología anómala, presentado como BBS ochentosa interactiva.

**[→ Abrir el nodo](https://TU-USUARIO.github.io/herculaneum-299/)**

## Qué es esto

Once archivos secuenciales que desarrollan un marco teórico integrando:

- Mecánica cuántica relacional y pasado observador-dependiente
- Soluciones relativistas de viaje temporal
- Teoría de la información y canal retrocausal
- Cerebro como amplificador cuántico-macroscópico
- Manifestaciones físicas transitorias
- Lectura de testimonios fringe bajo el marco
- La Singularidad como nodo topológico denso

Cada archivo distingue explícitamente entre elementos establecidos en literatura revisada, especulación apoyada en marcos teóricos existentes, y conjeturas originales del marco.

Los seis bloques marcados **◆◆◆ TESIS ORIGINAL ◆◆◆** contienen las formulaciones propias del nodo investigador.

## Cómo se usa

- **Desktop:** flechas ↑↓ para navegar, ENTER para abrir, ←→ para moverse entre archivos, ESC para volver al menú
- **Mobile:** tap en los archivos, botones de navegación abajo, swipe horizontal entre archivos
- La secuencia está diseñada para lectura ordenada 001 → 011

## Tecnología

Un solo archivo HTML autocontenido. Sin build, sin dependencias instalables, sin backend. Solo Google Fonts (VT323) por CDN.
